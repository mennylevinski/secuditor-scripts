# Secuditor Scripts

A partial collection of open source diagnostic scripts for Windows, used by the Secuditor project. All scripts in this repository are standalone, and some may require the installation of third party packages.

For the full application, visit [Secuditor Free](https://github.com/secuditor/secuditor-free) which is distributed separately as closed source freeware.

<br>

- `sp_gateway_detection.py` – Detects the local network's default gateway and public IP.
- `sp_installed_apps.py` – Generates a list of applications installed on the device.
- `sp_network_settings.py` – Presents the device's network adapters.
- `sp_shared_folders.py` – Presents a mapping of shared folders (Requires admin Permissions).
- `sp_system_audit.py` – Presents a detailed report of the system's settings and inventory.
